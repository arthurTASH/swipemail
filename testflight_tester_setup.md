# SwipeMail Tester Setup

## Before you start
- Install the latest TestFlight build.
- Use a Google account that has been added as an approved OAuth test user.
- Prefer an inbox with several unread primary messages so the swipe flow is easy to exercise.

## Recommended setup
1. Confirm the account can complete Google sign-in in the app.
2. Ensure there are at least a few unread messages in the Primary inbox category.
3. Keep network available for the main flow first, then test offline behavior separately.

## Useful edge cases
- No unread primary messages
- Network disabled before trying a swipe action
- Relaunch after sign-in
- Disconnect from Settings
- Exit and Resume from the drawer/settings flow
